## Gestor de tareas - Esqueleto Flask

Aplicación Flask mínima ya preparada para arrancar en modo desarrollo.

### Requisitos

- Python 3.10 o superior
- `pip` instalado

### Instalación

```bash
pip install -r requirements.txt
```

### Ejecución en desarrollo

```bash
python app.py
```

Luego abre en el navegador:

- `http://127.0.0.1:5000/`
